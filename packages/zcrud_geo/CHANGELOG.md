# Changelog

All notable changes to `zcrud_geo` are documented in this file.

## 0.1.0

Initial public release.

- Geo fields for zcrud: a neutral `ZGeoPoint`/`ZGeoShape` value model and a `ZGeoFieldWidget` served through `ZWidgetRegistry`.
- Part of the [zcrud](https://github.com/zakarius-dev/zcrud) monorepo (14 packages, one declarative CRUD engine).
- Published under the MIT license.
