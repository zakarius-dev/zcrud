# Changelog

All notable changes to `zcrud_export` are documented in this file.

## 0.1.0

Initial public release.

- Tabular export for zcrud: turn a neutral render request (columns and rows from `zcrud_core`) into Excel (`.xlsx`) or PDF bytes via Syncfusion.
- Part of the [zcrud](https://github.com/zakarius-dev/zcrud) monorepo (14 packages, one declarative CRUD engine).
- Published under the MIT license.
